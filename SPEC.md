# 賃貸vs購入コスト比較 - 生涯コストを無料シミュレーション - 技術仕様書

## 概要

**サービス名**: Rental Vs Buy Advisor
**バージョン**: 1.3.0
**更新日**: 2026-06-06
**URL**: https://appadaycreator.com/rental-vs-buy-advisor/

年齢・年収・家族構成から賃貸と住宅購入の生涯コストを比較。ローン返済・管理費・売却想定価格を含めた総合判断を無料で診断します。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## v1.5.0 改善内容（2026-06-10）

### P1 実装（UX改善・機能追加）

1. **入力フォームへのplaceholder属性追加（M2）**
   - 6個のnumber input要素に`placeholder`属性を追加
   - 例：年齢→"35"、年収→"600万円"、家賃→"12万円"、金利→"1.5%"
   - ユーザーが期待される値の形式を一目で理解でき、入力ストレスを軽減
   - aria-label属性も追加してアクセシビリティを向上

2. **計算履歴の自動保存・復元機能（M3）**
   - 計算実行時に結果を`localStorage`の`rentalVsBuyHistory`に自動保存
   - 履歴形式：`{ timestamp, age, income, rent, price, rate, downpayRate, period, family, rentTotal, netBuyCost, verdict }`
   - 最大10件の履歴を保持（新しい順）
   - 結果セクション下に「📜 過去の計算履歴」セクションを表示
   - 履歴カードから復元ボタンをクリックで入力値を復元し、自動再計算
   - 履歴クリアボタンで確認ダイアログ付きで削除可能

3. **結果をテキスト形式でコピー（M10）**
   - 結果セクションに「📋 結果をコピー」ボタンを追加
   - 計算結果を整形テキスト形式でクリップボードにコピー
   - コピー内容：入力条件 + 計算結果 + URL
   - メール・チャット・ドキュメントへの貼り付けに対応

### 実装詳細

- **saveToHistory()**: 計算結果をlocalStorageに保存
- **displayHistory()**: ページロード時に履歴セクションを表示
- **restoreFromHistory()**: 履歴から入力値を復元して自動再計算
- **clearHistory()**: 履歴を全削除（確認ダイアログ付き）
- **copyResultText()**: 結果をテキスト形式でクリップボード出力

## v1.4.0 改善内容（2026-06-06）

### P1 実装（UX改善・機能追加）

1. **入力フォームのplaceholder・説明テキスト追加（M2）**
   - 6個のinput要素に`placeholder`属性を追加（年齢・年収・家賃・物件価格・金利・頭金割合）
   - 複雑な項目（金利・頭金割合・居住期間・ローン控除）に`?`ツールチップを追加
   - ツールチップ内容：「現在の相場は1.5〜3.0%程度」「10%以上が一般的」「短期（15年未満）だと売却損が出やすい」等
   - ユーザーが迷わずに正確な値を入力できるように改善

2. **localStorage自動保存・復元機能（M3）**
   - `saveToLocalStorage()`関数を実装し、入力値の変更時に自動保存
   - `loadFromLocalStorage()`関数でページ読み込み時に前回の入力値を自動復元
   - チェックボックス・セレクトボックスも保存対象に含める
   - ユーザーが複数回試算する際の入力負担を削減

3. **複雑な項目へのツールチップ充実（M8）**
   - 金利・頭金割合・居住期間・ローン控除・家族構成の横に`?`アイコンを追加
   - `title`属性でツールチップを実装（マウスホバー時に表示）
   - 初心者ユーザーが専門用語を理解した上で入力できるように支援

4. **印刷・CSV エクスポート機能追加（M10）**
   - 結果セクションに「📄 印刷・PDF保存」ボタンを追加（`window.print()`）
   - 「📊 CSV形式でダウンロード」ボタンを追加
   - CSV内容：入力値 + 計算結果 + 詳細内訳表
   - ファイル名形式：`rental-vs-buy-YYYY-MM-DD.csv`
   - ユーザーが結果を家族・FP・不動産会社と共有しやすく改善

## v1.3.0 改善内容（2026-06-06）

### P1 実装（バグ修正・UX修正）

1. **ヘッダーテキスト不可視バグの修正**
   - ページ最上部タイトルが白背景に白文字で不可視だった問題を修正
   - ヘッダーセクションに青紫グラデーション背景（`#1d4ed8 → #7c3aed`）を追加

2. **ローン計算ロジックの正確化**
   - 居住期間内のローン支払い（元本・利息）を月次で逐次計算するよう変更
   - 売却時残債を実際の元本残高から正確に算出（従来は `loanAmt * 0.5` の固定値）
   - 実質購入コスト = 頭金 + 諸費用 + 居住期間内利息 + 管理費 - 売却純益 - ローン控除

3. **グラフの大型化・見やすさ改善**
   - Canvas幅を最大300px → 最大550pxに拡大、高さも200px → 260pxに拡大
   - バーの上に金額ラベルを表示
   - グラフ下部に差額テキスト（「購入がX万円お得」など）を表示

### P2 実装（機能追加）

4. **頭金割合のカスタマイズ**
   - 頭金をスライダーで0〜30%（5%刻み）で設定可能に
   - 頭金変更に応じて月々返済額・総コストがリアルタイム更新
   - フォームに頭金額プレビュー（「頭金目安: X万円」）を表示

5. **住宅ローン控除の計算への反映**
   - チェックボックスで住宅ローン控除（年末残高×0.7%×13年間、上限455万円）を適用可能
   - 控除額を購入コスト内訳に「ローン控除効果（マイナス）」として表示
   - シェアURLに控除適用状態を含む（`taxDed`パラメータ）
